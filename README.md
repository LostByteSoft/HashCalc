# HashCalc ORIGINAL by jNizM
Calculate hash from string, hex or file via AutoHotkey

## Features
* Generate hash value from string, hex or file
* Secure-Salted function
* Verify hash
* Load md5 sha ... etc from a file

### Hash Functions
* CRC32
* MD2, MD4, MD5
* SHA-1
* SHA-256, SHA-384, SHA-512
* HMAC
